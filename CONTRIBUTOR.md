## 📝 Contribution Attribution Format

When acknowledging external contributors, the format used is:

```
| @<github-handle> | <Brief description of contribution> | #<PR number> |
```

Example:

| Contributor | Contribution | PR |
| :--- | :--- | :---: |
| @example-user | Fixed mobile nav overflow on small viewports | [#42](https://github.com/deepdeyiitgn/My-Portfolio/pull/42) |
