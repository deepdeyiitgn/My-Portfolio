## 🤝 External Contributors

Contributions are listed in repository history. Pull request authors are credited automatically in commit history and release notes.

If you have contributed a meaningful improvement via a merged PR, your GitHub handle will be added here in the next release acknowledgement.

---

## 📝 Contribution Attribution Format

When acknowledging external contributors, the format used is:

```
- @<github-handle> — <Brief description of contribution>
```

Example:
```
- @example-user — Fixed mobile nav overflow on small viewports (#42)
```
