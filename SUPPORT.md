# Support

If you need help with this project, use the channels below.

## Usage & General Questions

- Open a GitHub Issue with clear steps and context.

## Security Issues

- Please do **not** open public issues for vulnerabilities.
- Follow the private reporting process in [SECURITY.md](./SECURITY.md).

## Contact Channels

- General/technical outreach: **team.deepdey@gmail.com**
- Security-specific reporting: **a@qlynk.me**
- Ticketed contact flow (web): **/contact** (auto ticket ID + mail client handoff)

## Before Opening an Issue

- Confirm the behavior on the latest version of the repository.
- Include reproduction steps, expected result, and actual result.
- Add screenshots or logs when relevant.
